How does object-oriented programming work?

Object-oriented programming is a style of programming in which code is organised into classes and methods - these are designed to manipulate any data entered in specific ways.

A class describes an object, which is defined by the programmer. Each class is designed to behave in a certain way within the system. A class object is an object that contains information about a class. An instance is an object belonging to a class. A class attribute is a value associated with an object (e.g. a name/string, a number/integer, float, boolean etc.). Class attributes are defined inside a class definition but outside any method so that the method is as flexible to call as possible. An embedded object is an object stored as an attribute of another object, meaning it inherits certain parts of it's parent object. A method is a function defined inside a class definition and is used in instances of that class.
