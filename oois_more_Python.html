Progress with Python

In this module we covered the following principles in Python programming:

Classes & Objects/ Class Functions and Methods: a class is defined by the programmer in Python and allows data and functions to be presented together. A class definition creates a new class object, which contains information about the programmer-defined class and allows new instances of the same type to be made. Attributes are values associated with the class object to keep it in the same state and class methods are used to modify the class's state. Classes are defined unsing the initiator function '__init__()'.

Inheritance: a programmer can use inheritance to define a new class which inherits the methods and properties from another class (i.e. a child class inherits from a parent class).

Encapsulation: is the term used to describe the concept of bundling together data and methods within one section of code (e.g. a class).

Polymorphism: is the term used to describe the fact that an object can take many forms. This allows the same methods to be used in a child class as in a parent class, for example (= method overriding).

Progress with SQL

The SQL techniques covered in this module were an extension of the Launching into Computer Science module and went into more depth in terms of accessing and manipulating tables of data. We practiced specifying constraints for data types in a database (e.g. using the DISTINCT and LIKE functions), and also updating and amending the data included therein using more elegant and efficient notation techniques (such as dot notation/ single expressions). We covered statements to create, alter and delete/ drop databases and also foreign key and primary keys. We also explored aggregate functions such as average, sum, count etc.
